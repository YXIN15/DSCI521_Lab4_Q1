## Ice Cream Flavors

We can make a choose some ice cream flavors and put them in a list.

``` r

flavors
```

## Chocolate Brands

We can also choose some brands of chocolate and put them in a vector.

``` r
chocolates <- c("Lindt", "Cadbury", "Ferrero")
chocolates
```

    ## [1] "Lindt"   "Cadbury" "Ferrero"
